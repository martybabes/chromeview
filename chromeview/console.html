                      FIRST THING

                      SET the Song Path (above console screen)

                      To create song files, view sample .txt files included with the package. Click around and experiment to discover features. Restart if you have real trouble.

                      Reserved Folders: (case sensitive)
                      - images (your images)
                      - media (your music mp3* only)
                      - programs (your saved progs)
                      - sheets (future-lead sheets)
                      - videos (your videos** - mp4 only)
                      All other folders are treated as song books and show up in the selection Tabs (max 10).
                      * '.mp3' file extension lowercase, file name identical to songfile name
                      eg. 'My Song.txt' My Song.mp3'
                      **Only put mp3 files in 'media' folder.

                      Show/hide list

                      Use Show/hide list to hide the tabbed songlist.
                      Click on a song in the list ot type a few letters in the textbox next to it to select a song.

                      Click to view selected song immediately.

                      The splash screen image can be changed in main settings. Place jpg files in the [yourpath]\images folder and name them 'splash.jpg', splash1.jpg, splash2.jpg or use the 'choose' for something else.

                      Program Tab

                      Use to add selected song to program
                      You may also insert Splash, Blank, Slide (image) screens or an MP4 video into the program.

                      Click to show the next item in the program list.
                      Clicking program line continues from that spot.
                      Splash screen is shown when program completed
                      [Reset] sets the line counter to zero so you can check the program and start over.

                      Click the [Edit] button to edit the program. (change the order or mode)
                      You can load a program from there.
                      To save: copy and paste into Notepad and save as an ordinary text file.

                      Mode TKI:
                      - [/]show title = 'T'
                      - [/]karaoke = 'K'
                      - [/]play immediately = 'I'
                      '_' = mode item switch off

                      Preview Tab

                      Display songs by clicking in the appropriate screen in the preview table OR...
                      Click to show next screen of a song = follows ORDER.
                      A line should be included in song .txt file eg. 'ORDER:123242' if song has choruses and bridges.
                      If 'show title' checked ('T' in program mode), then first screen is the title screen (else 1st screen in ORDER). If song has backing audio file, create screen timings (under Times tab), add the 'PLAY:...' line to the song file, use the play button to start the music and watch screens change automatically.

                      Times Tab

                      Songs with a backing track (karaoke mode) get there screen change times listed here. You can click a line to change the songs position while playing.
                      You can also create/edit a list of timings here for your song.

                      Settings Tab

                      Click to toggle settings view.
                      Here you can change various song viewer parameters.
                      You can save them as defaults for next time you run this app. (See 'FIRST THING' for setting root folder)

                      'CI:' Builder enables you to build a special set of display instructions to add to a verse in your song.(see 'Special Lyrics Commands' below)

                      Special Lyrics Commands

                      See sample song text files

                      Note the use of space and comma!

                      FB: (Foreground/Background)

                      FB:csscolor imagepath/imagefile.ext
                      FB:.cssclass imagepath/imagefile.ext

                      eg1:
                      FB:black images/xmas/palmbg.jpg
                      (Everybody Sing)
                      Hosanna. Blessed be the Rock!

                      (Look up css colours on the web)

                      eg2:
                      FB:.txtblackinwhite images/xmas/palmbg.jpg
                      (Everybody Sing)
                      (dot)colorclass found in CI Builder

                      CI: (Classes/Image)
                      Use CI: Builder (under Settings)
                      All parameters are predefined CSS classes (except for optional image file)
                      Note use of comma
                      MUST be single line in text file.

                      CI:bg-color text-position,
                          text-layout text-color font-height font-width,
                          '#' or imagepath/imagefile.ext

                      eg:
                      CI:bgimage midcenter, txtcenter txtblackinwhite fontYnormal fontXnormal,black images/xmas/palmbg.jpg
                      Hosanna. Blessed be the Rock!
